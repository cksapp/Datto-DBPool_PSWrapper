    Remove-DBPoolBaseURI

    SYNOPSIS

    Removes the DBPool base URI global variable.

    SYNTAX

    Remove-DBPoolBaseURI [-WhatIf] [-Confirm] [<CommonParameters>]

    DESCRIPTION

    The Remove-DBPoolBaseURI cmdlet removes the DBPool base URI global variable.

    EXAMPLES

    EXAMPLE 1

    Remove-DBPoolBaseURI

    Removes the DBPool base URI global variable.
