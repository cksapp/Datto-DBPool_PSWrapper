# Get-DBPoolBaseURI

## SYNOPSIS
Shows the DBPool base URI global variable.

## SYNTAX

```
Get-DBPoolBaseURI [<CommonParameters>]
```

## DESCRIPTION
The Get-DBPoolBaseURI cmdlet shows the DBPool base URI global variable value.

## EXAMPLES

### EXAMPLE 1
```

```

Shows the DBPool base URI global variable value.
