# Remove-DBPoolApiKey

## SYNOPSIS

Removes the DBPool API Key global variable.

## SYNTAX

```PowerShell
Remove-DBPoolApiKey [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

The Remove-DBPoolAPIKey cmdlet removes the DBPool API Key global variable.

## EXAMPLES

### EXAMPLE 1

```PowerShell
Remove-DBPoolAPIKey
```

Removes the DBPool API Key global variable.
